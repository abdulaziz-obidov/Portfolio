- 🌑 **Dark Mode** — ko'zni toliqtirmaydigan zamonaviy to'q rangli dizayn
- 🎨 **Glassmorphism UI** — shisha effektli kartochkalar va komponentlar

- 📊 **Animated Skill Bars** — ko'nikmalar darajasini ko'rsatuvchi animatsiyali chiziqlar

- 📬 **Contact Form** — interaktiv aloqa formasi

## 🚀 Ishga Tushirish (Getting Started)

Bu loyiha hech qanday server yoki installatsiya talab qilmaydi.

### Usul 1 — To'g'ridan-to'g'ri ochish

```bash
# Faylni brauzerda oching
index.html → O'ng tugma → "Open with" → Chrome/Edge/Firefox
```

### Usul 2 — VS Code Live Server

```bash
# 1. VS Code-da papkani oching
code .

# 2. Live Server extension o'rnating (agar yo'q bo'lsa)
# 3. index.html → O'ng tugma → "Open with Live Server"
```

### Usul 3 — Python HTTP Server

```bash
# Python 3 bilan
python -m http.server 8080

# Brauzerda oching
http://localhost:8080
```
